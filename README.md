# 🌍 Real-Time AQI Monitoring System (Ajmer)

## 📊 Overview

Developed a real-time Air Quality Monitoring System using API data to track pollution levels and provide actionable health insights.

## 🧰 Tech Stack

* Python (API integration)
* Power BI (Dashboard)
* OpenWeatherMap API (Air Pollution Data)

## 🚀 Features

* Fetches real-time and historical AQI data
* Tracks pollutants like PM2.5, PM10, CO, NO2, O3
* Interactive dashboard with AQI trends
* Health advisory based on AQI levels

## 📈 Key Insights

* Identified peak pollution periods using time-series analysis
* Compared pollutant contributions to AQI
* Highlighted health risks based on AQI categories

## ⚙️ How to Run

1. Add your API key in `fetch_aqi.py`
2. Run the script to fetch data
3. Load CSV into Power BI
4. Open `.pbix` file to view dashboard

## 🎯 Outcome

Built a real-world data pipeline integrating API, data processing, and visualization to monitor air quality effectively.
